# OfficeFlow Frontend

OfficeFlow 前端项目，使用 Vue3 + Element Plus + Pinia + Vue Router + Axios + ECharts。

## 启动

安装依赖：

```powershell
pnpm install
```

启动开发服务：

```powershell
pnpm dev
```

构建：

```powershell
pnpm build
```

默认访问：

```text
http://localhost:5173
```

后端接口统一通过网关访问：

```text
http://localhost:9000
```
