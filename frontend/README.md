# OfficeFlow Frontend

前端项目计划使用 Vue3 + Element Plus + Pinia + Vue Router + Axios + ECharts。

建议创建方式：

```powershell
pnpm create vite . --template vue
pnpm install
pnpm add element-plus @element-plus/icons-vue axios pinia vue-router echarts
pnpm dev
```

前端统一通过网关访问后端接口：

```text
http://localhost:9000
```
